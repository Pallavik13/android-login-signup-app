# Android Login & Signup App

## 📌 Description
A simple Android application that demonstrates basic user authentication flow using login and signup functionality. Existing users can log in successfully, while new users are redirected to the signup page to create an account.

## ✨ Features
- User Login functionality
- User Registration (Signup)
- Input validation for user credentials
- Navigation between Login and Signup screens
- Simple and user-friendly UI

## 🛠️ Tech Stack
- Java
- Android SDK
- XML (UI Design)

## 🚀 How to Use
1. Open the app
2. Enter login credentials
3. If the user exists → Login successful
4. If not → Redirected to Signup page
5. Create a new account and log in

## 📁 Project Structure
This repository currently contains the project as a ZIP file due to device limitations.  
The complete source code can be accessed by downloading and extracting the ZIP file.
